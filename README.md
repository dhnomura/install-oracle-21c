At least 1 GB RAM for Oracle Database installations. 2 GB RAM recommended.

## OS

Red Hat Enterprise Linux 8.2: 4.18.0-193.19.1.el8_2.x86_64 or later

## Server Configuration

/tmp - 1GB
SWAP - 2GB - 16GB

Storage Check List - 50 GB
